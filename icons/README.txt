App Icons for Fitness Tracker

This directory should contain the following icon files for the Progressive Web App:

- icon-72x72.png (72x72 pixels)
- icon-96x96.png (96x96 pixels)
- icon-128x128.png (128x128 pixels)
- icon-144x144.png (144x144 pixels)
- icon-152x152.png (152x152 pixels)
- icon-192x192.png (192x192 pixels)
- icon-384x384.png (384x384 pixels)
- icon-512x512.png (512x512 pixels)

These icons should be created using the app's logo with appropriate padding and sizing.
The icons should be in PNG format with a transparent background.

For the best results:
1. Use a simple, recognizable design
2. Ensure the icon is clearly visible at small sizes
3. Use the app's primary color (#4CAF50) as the background
4. Add appropriate padding around the icon (about 10% of the icon size)
5. Test the icons on different devices and backgrounds

You can generate these icons using tools like:
- https://app-manifest.firebaseapp.com/
- https://realfavicongenerator.net/
- Adobe Illustrator or Photoshop
- Sketch or Figma
